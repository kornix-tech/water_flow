# Roadmap web-режима SoilFlow/PFLOTRAN

1. Web-редактор XLSX-параметров с формами, единицами измерения и валидацией.
2. Сравнение двух сценариев с общей таблицей различий и графиками.
3. Очередь расчётов с приоритетами и ограничениями ресурсов.
4. Авторизация пользователей и роли для командной работы.
5. Reverse proxy + HTTPS для VDS.
6. PostgreSQL вместо SQLite при появлении многопользовательского режима.
7. Redis/Celery/RQ для тяжёлой очереди jobs, если одного контейнера станет мало.
8. Разделение frontend/backend compose stack при необходимости масштабирования.
9. Экспорт отчёта PDF/DOCX по расчёту и визуализации.
10. Сценарная база проектов с историей запусков.
